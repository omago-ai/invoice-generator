# Invoice Generator

Free, open-source invoice generator. Create professional invoices and download as PDF.

**Live:** [invoice.omago.ai](https://invoice.omago.ai)

## Features
- Beautiful, Stripe-quality PDF output
- Logo upload & brand color picker
- Multi-currency support
- Fully client-side — no data stored
- No signup required

## Development
Open `index.html` in a browser. No build step needed.

## Deployment
Deployed via Cloudflare Pages from this repository.
